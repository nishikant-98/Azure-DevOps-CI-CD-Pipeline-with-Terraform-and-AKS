# 🚀 End-to-End CI/CD Pipeline Project (DevOps)

This project demonstrates a **complete CI/CD pipeline** from code commit to production deployment using modern DevOps tools.

---

## 📌 📖 Project Overview

This project simulates a real-world DevOps workflow:

👉 Developer pushes code → CI/CD pipeline triggers → Build → Test → Analyze → Containerize → Deploy → Production 🚀  

---

## 🧩 Tech Stack

- **Application**: Node.js (Express)
- **CI/CD Tools**: GitHub Actions, Jenkins
- **Code Quality**: SonarQube
- **Containerization**: Docker
- **Container Registry**: Docker Hub / AWS ECR
- **Orchestration**: Kubernetes (EKS / AKS / Local)
- **Infrastructure as Code**: Terraform

---

## 🏗️ Architecture Flow

Developer → Git → CI Server → Build/Test → Docker → Registry → QA → Production
